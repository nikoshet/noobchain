Basic Functionality Of The Web Application:
Functionality Page Info
Create A Transaction Home User inserts his public key, the public key of the receiver, and the amount of NBCs to transfer. After submitting the transaction, a message will pop up that will inform him if the transaction was made successfully or if any errors occured.
View Transactions Made In Last Verified Block Wallet There is a table that contains information about the transactions made in the last verified block of the blockchain.
Balance Statistics Wallet An area that shows the available balance of the current node, as well as the total amount of verified NBCs received and sent.
View Blockchain Blockchain An area that shows the current blockchain.
CPU / Memory Usage Home An area shows the current CPU and Memory usage of the current node, in order to monitor the system's performance.
Application's Info About A page that includes all the necessary info for the functionality of the Blockchain (Noobchain) Application, as well as the REST API implemented.
